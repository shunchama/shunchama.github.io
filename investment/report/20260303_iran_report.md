# 2026年3月以降のイラン情勢におけるグローバル市場と株価の構造的変容

## 「オペレーション・エピック・フューリー」後の地政学的ブラック・スワンと経済的衝撃

---

## 序論：21世紀最大の地政学的転換点

2026年2月28日、米国とイスラエルによる共同軍事作戦**「オペレーション・エピック・フューリー（Operation Epic Fury）」**が発動され、イランの最高指導者アリ・ハメイニが死亡した。これは21世紀の金融市場における最大の「ブラック・スワン」イベントとなり、グローバルなエネルギー供給網、通貨制度、そして各国の中央銀行による金融政策の前提条件を根本から覆している。

---

## 1. 紛争の背景とイラン国内の統治空白

2025年から続く大規模な抗議デモにより、イラン体制は既に末期的な脆弱性を示していた。最高指導者の急逝により、現在イランは「暫定評議会」による統治下にあるが、革命防衛隊（IRGC）残存勢力による報復行動など、内戦に近い不確実性が継続している。

### イラン国内の主要指標（2026年3月時点）

| 指標             | 状況・予測                               |
| :--------------- | :--------------------------------------- |
| **統治形態**     | 3名の暫定評議会による暫定統治            |
| **経済成長率**   | 2026年予測：大幅なマイナス成長           |
| **インフレ率**   | 年率60%に接近                            |
| **主要輸出施設** | ハルク島（Kharg Island）施設に損傷の報告 |

## 2. ホルムズ海峡の封鎖とエネルギー供給ショック

世界の石油供給の約20%（日量約2,000万バレル）が通過する**ホルムズ海峡**の閉鎖宣言は、ロジスティクス面での致命的な機能不全を招いている。海運大手（Maersk等）は喜望峰ルートへの変更を余儀なくされ、輸送日数は10〜15日延長、コストは劇的に上昇した。

## 3. グローバル株式市場の動向：セクター別の明暗

2026年3月2日、世界中の証券取引所は「リスクオフ」一色となった。しかし、その内実を精査すると明確な勝者と敗者が分かれている。

### 3.1 米国市場：防衛とAIの二極化

- **上昇（防衛・エネルギー）**: ノースロップ・グラマン（+5.9%）、パランティア（+5.8%）。有事における物理的・デジタル防衛需要が急騰。
- **下落（ハイテク）**: エヌビディア（-1.3%）。供給網寸断と金利上昇懸念が重石。

### 3.2 日本市場：エネルギー脆弱性の露呈

日本は中東依存度の高さからアジアで最も激しい売りを浴び、日経平均は一時**1,700円超の下落**を記録。

- **壊滅**: 石油・石炭製品（ENEOS等）、空運（JAL等）。
- **乱高下**: 海運（日本郵船等）、総合商社。

## 4. 金融政策と安全資産のパラダイムシフト

### スタグフレーションの懸念

エネルギー価格高騰によるコストプッシュ・インフレが再燃。Fed（米連邦準備制度）は利下げ観測を後退させ、日本銀行は円安（1ドル=157円台）と物価高の板挟みとなり、追加利上げ判断を保留せざるを得ない苦境に立たされている。

### 安全資産の定義変化

今回の危機では、伝統的な「有事の円買い」が機能せず、円安が進行。代わりに**金（ゴールド）**と**米ドル**への一極集中が加速している。

| 資産クラス           | 騰落率（3/2-3/3）  | 投資家心理                                 |
| :------------------- | :----------------- | :----------------------------------------- |
| **スポット金**       | **+2.3% 〜 +5.2%** | 究極の安全資産（デベースメント・トレード） |
| **米ドル指数 (DXY)** | +1.1% 〜 +1.8%     | 世界最大の流動性への逃避                   |
| **日本円**           | **下落 (157円台)** | エネルギー輸入コスト増による経常赤字懸念   |

## 5. 長期シナリオと投資家への提言

専門家は、今後の推移について3つのシナリオを提示している。

1.  **限定的エスカレーション（基本）**: 米軍により海峡が数週間で解放。市場は早期回復へ。
2.  **スタグフレーションの定着（悲観）**: 紛争が長期化し、原油100ドル超が常態化。世界的なベアマーケットへ。
3.  **イランの民主的再統合（楽観）**: 暫定評議会が親欧米派と協調。イラン資源が市場に復帰し、強力なデフレ的刺激に。

> **戦略的視点**: 投資家は、地政学リスクをヘッジするための「防衛・エネルギー」の保有、サイバーセキュリティ銘柄への注視、そして円に頼らない「金・外貨」への分散が不可欠である。

## 結論

2026年3月以降のイラン情勢は、世界経済に「不確実性という名のコスト」を恒久的に追加した。ハメイニ体制の崩壊は、短期的にはパニックを招いたが、長期的には凍結されていた中東の地政学的力学を動かす「痛みを伴う授業」となる。投資家は、物理的な船舶の動静と中央銀行の判断、その両方を冷徹に見極める必要がある。
