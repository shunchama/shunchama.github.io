# 8964天安門事件(Tiananmen Square Massacre)

1989年に発生した天安門事件は、現代中国の政治的軌跡を決定づけ、国際社会における中国の立ち位置を根本から変容させた歴史的転換点である。本報告書では、4月の胡耀邦死去から6月4日の武力鎮圧、そしてその後の国際的な影響に至るまでの経緯を詳細に分析する。

## 第一章 背景：改革開放の矛盾と言論の胎動

1980年代後半の中国は、鄧小平が進める「改革開放」政策の過渡期にあり、経済の自由化と政治の硬直性の間で深刻な摩擦が生じていた。

### 1.1 経済的混迷と政治改革の遅滞

1988年にはインフレ率が18.5%に達し、都市住民の生活は困窮。これに加え、共産党幹部の子弟（太子党）による経済的特権の独占（官倒）が深刻な社会問題となった。

| 構造的要因     | 具体的な事象・現象                     | 社会への影響                               |
| :------------- | :------------------------------------- | :----------------------------------------- |
| **経済的要因** | 二重価格制の弊害、官倒、急激なインフレ | 都市労働者や学生の生活困窮、不公平感の増大 |
| **政治的要因** | 政治改革の停滞、言論統制、保守派の台頭 | 知識層や学生による民主化・透明性への渇望   |
| **社会的要因** | 指導部家族の腐敗、官僚主義の蔓延       | 政府に対する信頼の失墜、改革の停滞への不安 |

## 第二章 運動の勃発：1989年4月15日 - 4月22日

事件の直接的な導火線となったのは、1989年4月15日の**胡耀邦**・前総書記の死であった。

### 2.1 追悼から抗議へ

胡耀邦の死去直後から、学生たちは「民主化に理解のある指導者」として彼を追悼し、同時に「七項目要求」を政府に突きつけた。

### 2.2 七項目要求の核心的意義

要求項目は単なる追悼を超え、国家の構造的改革を迫るものであった。

| 要求項目               | 詳細内容                             | 政治的背景・意図                           |
| :--------------------- | :----------------------------------- | :----------------------------------------- |
| **胡耀邦の再評価**     | 彼の「民主化」への姿勢を肯定すること | 1987年の失脚が誤りであったことを認めさせる |
| **反自由化運動の否定** | 「反精神汚染」運動などの完全停止     | 保守派による思想統制の解除を求める         |
| **指導者の財産公開**   | 指導者とその家族の資産を公表         | 官倒（腐敗）を可視化し、根絶する           |
| **言論・報道の自由**   | 民間新聞の発行許可と言論の自由       | 党による情報独占を打破する                 |
| **教育予算の増額**     | 教育費拡大と知識人の待遇改善         | 恩恵から疎外された知識層の救済             |
| **デモ規制の撤廃**     | 北京市のデモ抑制条例の取り消し       | 結社・集会の自由という権利の確立           |
| **公正な報道の実施**   | 抗議活動を真実に即して報道           | 運動が「動乱」として歪曲されるのを防ぐ     |

## 第三章 指導部の亀裂と「四・二六社説」：4月下旬

共産党指導部は、趙紫陽を中心とする**穏健派**と、李鵬を中心とする**強硬派**に分かれ深刻な対立に陥った。

### 3.1 鄧小平の決断と動乱の規定

4月26日、『人民日報』に社説**「旗幟鮮明に動乱に反対せよ（四・二六社説）」**が掲載。運動を「周到に計画された陰謀」と断じた。

### 3.2 四・二七大規模デモの衝撃

「動乱」規定に反発した学生たちは、4月27日に大規模デモを強行。警察の阻止線を突破し、参加者は100万人に達した。

## 第四章 絶食抗議とゴルバチョフ訪中：1989年5月

5月13日、ソ連のゴルバチョフ訪中を前に、学生たちは天安門広場での**絶食抗議**を開始。

### 4.1 北京の「無政府状態」

絶食抗議に同情した市民、労働者、公務員までもがデモに加わり、北京は事実上の無政府状態に陥った。

| 5月中旬の主要動向             | 事象の詳細                       | 政治的・社会的影響                             |
| :---------------------------- | :------------------------------- | :--------------------------------------------- |
| **ゴルバチョフ訪中 (5/15)**   | 30年ぶりの中ソ首脳会談           | 歓迎式典が空港に変更されるという外交的屈辱     |
| **100万人デモ (5/17-18)**     | 市民、労働者、公務員が参加       | 運動が「全人民的」な広がりを見せ、党の威信失墜 |
| **趙紫陽の最後の訴え (5/19)** | 広場の学生を訪問し絶食中止を懇願 | 「来るのが遅すぎた」という言葉と共に失脚       |

## 第五章 戒厳令の発令と膠着：1989年5月19日 - 6月2日

5月20日、北京市内に**戒厳令**が正式に布告。しかし、市民によるバリケードが行く手を阻み、軍の進軍は一時的に停滞した。

### 5.1 民主の女神

5月30日、広場に高さ約10メートルの**「民主の女神」**像が設置され、自由への渇望を象徴した。

---

## 第六章 武力鎮圧（清場）：1989年6月3日 - 6月4日

6月3日深夜から4日未明にかけて、戒厳部隊による武力行使が開始された。

### 6.1 木樨地（ムキシチ）における流血

進軍する第38集団軍は、路上を封鎖する市民に対し実弾を使用。最も激しい流血が発生した。

| 衝突地点           | 状況と被害                           | 軍の行動                           |
| :----------------- | :----------------------------------- | :--------------------------------- |
| **木樨地**         | 市民がバリケード構築。最も激しい流血 | 銃撃により死傷者多数。救急車も標的 |
| **西単・長安街**   | 広場へのメインルート。市民が軍を阻止 | 装甲車による強行突破と実弾発砲     |
| **天安門広場周辺** | 4日午前1時、軍が包囲完了             | ビルや群衆に向けた威嚇射撃と発砲   |

## 第七章 犠牲者数と隠蔽の構造

犠牲者数については、政府発表と国際社会の推計で大きな乖離がある。

| 報告主体                    | 推計死者数     | 内容と根拠                                 |
| :-------------------------- | :------------- | :----------------------------------------- |
| **中国政府 (1989年6月)**    | 200〜300人余り | 学生は36人のみ、軍・警察数十人を含むと主張 |
| **中国公安省 (内部資料)**   | 931人          | 北京での死者は523人。地方都市でも死を確認  |
| **中国赤十字社 (初期推計)** | 約2,600人      | 事件直後の病院調査。後に撤回させられる     |
| **イギリス機密公電**        | 約10,000人     | 中国指導部近辺の情報源に基づく大使の報告   |

## 第八章 鎮圧後の展開と「戦車の男」

6月5日朝、長安街で一人の男が戦車の列の前に立ちふさがる**「戦車の男（タンクマン）」**事件が発生。非暴力の抵抗の象徴となった。

### 8.1 粛清と新体制

鎮圧後、趙紫陽は失脚し、後任に**江沢民**が就任。「安定がすべてに優先する（穏定圧倒一切）」の方針のもと、政治的締め付けが強化された。

## 第九章 国際的孤立と日本の外交的役割

西側諸国が制裁を科す中、日本は独自の判断を下した。

- **海部俊樹内閣の決断**: 中国の完全な孤立はアジアの不安定化を招くと懸念。
- **円借款の再開**: 1990年のヒューストン・サミットで円借款の再開を表明。中国の国際社会復帰の足がかりとなった。

---

## 第十章 結論：天安門事件の歴史的遺産と現在

1989年天安門事件は、中国共産党が「統治の正当性」の源泉を、民主的な手続きから**「経済成長」**と**「秩序維持」**へと完全にシフトさせた事件であった。

当時の学生が求めた「腐敗の根絶」や「言論の自由」は、未だ解決されない構造的課題として伏在している。現在の高度な監視社会システムは、「1989年の再来」を防ぐための巨大な予防装置としての側面を持っている。
